# ドンパチゲーム
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
主人公の犬がサバを敵である猫に投げつけるゲーム

## 操作方法
* 右矢印を押すとキャラが右方向に移動
* 左矢印を押すとキャラが左方向に移動
* SPACEキーを押すとサバを飛ばす
* F1を押すと現在のFPSがわかる

## クリア条件
* 敵を倒す

## ゲームの実装
###共通基本機能
* 主人公キャラクターに関するクラス
* 敵にクラスに関するクラス
* 弾に関するクラス

### 担当追加機能
* 白い弾からサバの画像に変えた
* 赤い四角形のプレイヤーを犬の画像に変えた
